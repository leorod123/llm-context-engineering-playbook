# Promotion And Closure Contract

## Core Rule

Discoverable does not mean trusted.

Indexed docs can still be restricted.

## Strong Promotion Requirements

A doc or project may become `bot_usage: allowed` only when:

- `semantic_status: validated`
- `evidence_status: validated`
- `validation_state` is `semantically_validated` or `runtime_validated`
- critical validation gaps are clear
- cross-system impact is resolved or explicitly not applicable
- required contracts are present or explicitly not applicable

## Closure Rule

Project closure requires a readiness record.

If evidence is partial, keep the project restricted.
