# Raw Handling Contract

## Purpose

Raw history is useful. It is also dangerous as default context.

## Rule

Put raw notes, chat exports, old plans, and migration snapshots under `raw/`.

The registry excludes `raw/` from default discovery.

## Use Raw Material When

- exact wording matters
- migration lineage matters
- a reconstruction audit needs original context

## Do Not Use Raw Material When

- answering normal implementation questions
- selecting default bot context
- promoting project maturity

Raw material can support investigation, but it does not become authority by
being old or detailed.
